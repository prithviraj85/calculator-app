# Calculator App 🧮

A basic calculator built using HTML, CSS, and JavaScript with operator validation, percentage handling, and error management.

## Demo

---

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Operator validation (prevents duplicate operators)
- Backspace and clear (AC)
- Percentage support (`% → /100`)
- Decimal support
- Result up to 3 decimal places
- Error handling (invalid input)
---

## Tech Stack
- HTML
- CSS
- JavaScript (DOM, Events, String methods,)
---

## Key Learnings
- DOM manipulation
- Event handling
- Input validation
---

## Limitations
- Uses `eval()` (not safe for production)
- Simplified `%` logic
- Some edge cases still possible
- Error handling `try-catch`
---

## Future Improvements
- Keyboard support
- Better validation (no invalid expressions)
- Replace `eval()` with custom logic
- Real calculator % logic
- Add history feature
- Error handling `try-catch`
